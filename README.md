# Algo Trading Bot

## Folders
* cppclient is the all API material
* TestCppClient is actual client that creates the socket connect

`$ git pull https://github.com/zhangc97/Algo_Trading_Bot-Linux-`

## Usage 
```
cd TestCppClient
make clean
make 
./TestCppClient
```
